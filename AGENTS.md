# API Integration Hub - Agent Instructions

This is a portfolio project for Jazhem Hamid.

## Goal
Build a polished developer-tool dashboard that showcases API integration, backend route handlers, webhooks, validation, and clean UI.

## Tech Stack
- Next.js App Router
- TypeScript
- Tailwind CSS
- Route Handlers
- Zod validation
- Public external APIs where possible

## Rules
- Keep the UI dark, modern, and portfolio-ready.
- Use reusable components.
- Do not expose secrets.
- Use .env.example for future API keys.
- Keep external API calls server-side when appropriate.
- Validate request bodies with Zod.
- Add loading, empty, success, and error states.
- Run npm run build and npm run lint after major changes.
